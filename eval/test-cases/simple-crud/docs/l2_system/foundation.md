# SimpleNote 技術基盤

## 1. 用語集

| 用語 | 説明 |
|------|------|
| メモ | ユーザーが作成するテキストデータ。タイトルと本文を持つ |

---

## 2. 技術選定

### 2.1 フロントエンド

| 項目 | 選定技術 |
|------|---------|
| フレームワーク | React 18 |
| 言語 | TypeScript |
| スタイリング | TailwindCSS |

### 2.2 バックエンド

| 項目 | 選定技術 |
|------|---------|
| ランタイム | Node.js 20 |
| フレームワーク | Express |
| ORM | Prisma |
| データベース | SQLite |

---

## 3. API設計方針

### 3.1 エンドポイント

| メソッド | パス | 説明 |
|---------|------|------|
| GET | /api/notes | メモ一覧取得 |
| POST | /api/notes | メモ作成 |
| DELETE | /api/notes/:id | メモ削除 |

### 3.2 エラーレスポンス形式

```json
{
  "error": {
    "code": "NOTE_NOT_FOUND",
    "message": "指定されたメモが見つかりません"
  }
}
```
