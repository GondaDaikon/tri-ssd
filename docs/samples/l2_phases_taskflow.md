# TaskFlow フェーズ定義

## 1. フェーズ概要

TaskFlowは以下の3フェーズで段階的にリリースします。

| フェーズID | フェーズ名 | 目的 | 対象ユーザー |
|-----------|-----------|------|-------------|
| PH-20251126-001 | MVP | コア機能の検証 | 社内テスター |
| PH-20251126-002 | β版 | 外部ユーザーでの検証 | 限定招待ユーザー |
| PH-20251126-003 | 正式版 | 一般公開 | 全ユーザー |

---

## 2. Phase 1: MVP（Minimum Viable Product）

### 2.1 フェーズ情報

```yaml
id: PH-20251126-001
name: MVP
goal: 個人向けタスク管理の基本機能を提供し、価値仮説を検証
exit_criteria:
  - タスクCRUD機能が動作する
  - ユーザー認証が機能する
  - 社内テスター10名がデイリーで利用
related_nfr_ids:
  - NF-20251126-001  # レスポンス性能
  - NF-20251126-003  # データ暗号化
  - NF-20251126-004  # 認証セキュリティ
```

### 2.2 対象機能一覧（記載順＝実装順）

| 順序 | 機能ID | 機能名 | 対応要件 | 概要 |
|------|--------|--------|----------|------|
| 1 | F-20251126-001 | ユーザー登録・ログイン | REQ-20251126-006 | メール/パスワードで新規登録・ログイン |
| 2 | F-20251126-002 | タスク登録 | REQ-20251126-001 | タイトル・期限・優先度を指定してタスク作成 |
| 3 | F-20251126-003 | タスク一覧表示 | REQ-20251126-002 | 登録済みタスクをリスト形式で表示 |
| 4 | F-20251126-004 | タスク完了操作 | REQ-20251126-003 | タスクを完了状態にマーク |
| 5 | F-20251126-005 | タスク編集 | REQ-20251126-004 | 既存タスクの内容を編集 |
| 6 | F-20251126-006 | タスク削除 | REQ-20251126-005 | 不要なタスクを削除 |

### 2.3 スコープ外

- プロジェクト管理機能
- チーム共有機能
- 検索・フィルタ機能
- 通知機能

### 2.4 Exit Criteria

- [ ] 全機能のE2Eテストがパス
- [ ] 社内テスター10名以上が1週間継続利用
- [ ] 重大バグ（P0）が0件
- [ ] フィードバックを収集し、Phase 2に反映

## 3. Phase 2: β版

### 3.1 フェーズ情報

```yaml
id: PH-20251126-002
name: β版
goal: 外部ユーザーでのユーザビリティ検証、プロジェクト管理機能の追加
exit_criteria:
  - 招待ユーザー100名が登録
  - NPS 30以上
  - 重大バグ0件
related_nfr_ids:
  - NF-20251126-001  # レスポンス性能
  - NF-20251126-002  # 同時接続
  - NF-20251126-005  # 可用性
  - NF-20251126-006  # バックアップ
  - NF-20251126-008  # アクセシビリティ
```

### 3.2 対象機能一覧（記載順＝実装順）

| 順序 | 機能ID | 機能名 | 対応要件 | 概要 |
|------|--------|--------|----------|------|
| 1 | F-20251126-007 | プロジェクト作成 | REQ-20251126-008 | タスクをグループ化するプロジェクトを作成 |
| 2 | F-20251126-008 | プロジェクトへのタスク紐付け | REQ-20251126-008 | タスクをプロジェクトに割り当て |
| 3 | F-20251126-009 | フィルタ機能 | REQ-20251126-007 | 期限・優先度でタスクを絞り込み |
| 4 | F-20251126-010 | キーワード検索 | REQ-20251126-007 | タスク名・説明文でキーワード検索 |
| 5 | F-20251126-011 | ダッシュボード | - | 今日・今週のタスク概要を表示 |

### 3.3 スコープ外

- チーム共有機能
- 通知機能
- 外部サービス連携

### 3.4 Exit Criteria

- [ ] 招待ユーザー100名以上が登録
- [ ] WAU（週間アクティブユーザー）50名以上
- [ ] NPSアンケート実施、スコア30以上
- [ ] 重大バグ（P0）が0件
- [ ] パフォーマンス要件（500ms以内）達成
